# 🎓 College Placement Analyzer

A beginner AI/Data Science project built using Python, Pandas, NumPy, Matplotlib, and Seaborn.

This project analyzes college placement data using preprocessing, feature engineering, statistical analysis, and visualizations.

---

# 🚀 Features

- 📥 CSV Dataset Loading
- 🧹 Data Cleaning
- 🔢 Encoding Categorical Data
- 📊 Correlation Analysis
- 🌡 Heatmap Visualization
- 📈 Histogram Visualization
- 📍 Scatter Plot Visualization
- 🏗 Employability Score Feature Engineering

---

# 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
